# CodeGeni Brand Assets

Official brand assets for the CodeGeni code review tool.

## Logo Files

- **logo-128.png** (128x128) - VS Code extension icon, favicons
- **logo-256.png** (256x256) - Small web usage, app icons
- **logo-512.png** (512x512) - Medium web usage, social media
- **logo-1024.png** (1024x1024) - High-resolution for print, large displays

## Brand Identity

**Colors:**
- Primary Orange: `#FF6B35` / `rgb(255, 107, 53)`
- Background Dark: `#2D2D2D` / `rgb(45, 45, 45)`

**Mascot:**
- Friendly genie character representing intelligent code assistance
- Surrounded by code symbols (`</>`, `{}`) representing multi-language support

**Typography:**
- Brand name: "CodeGeni" in bold, modern sans-serif
- Consistent orange color matching the mascot

## Usage Guidelines

### Do's
- ✓ Use on dark backgrounds for best contrast
- ✓ Maintain aspect ratio when resizing
- ✓ Ensure adequate padding around the logo
- ✓ Use PNG format to preserve transparency

### Don'ts
- ✗ Don't alter the colors
- ✗ Don't rotate or distort the logo
- ✗ Don't place on busy backgrounds
- ✗ Don't add effects like shadows or outlines

## File Generation

Brand assets are generated from the source icon using:
```bash
cd extensions/vscode-codegeni
python3 create_brand_assets.py
```

This creates multiple sizes optimized for different use cases.
